Kebijakan Privasi

Terakhir Diperbarui: 20 Januari 2026

1. Pengumpulan Data

Di GardaIdentitas, privasi Anda adalah prioritas utama kami. Kami menerapkan prinsip Data Minimization dan Local Processing.

  • Tidak Ada Pengumpulan Data Server: Kami tidak mengunggah, menyimpan, atau mentransfer foto identitas (KTP/SIM) Anda ke server mana pun.
  • Pemrosesan Lokal: Semua proses watermarking dilakukan sepenuhnya di dalam memori browser (Client-Side) perangkat Anda menggunakan teknologi HTML5 Canvas.
  • Data Volatile: Data gambar yang Anda pilih hanya ada di RAM saat tab browser terbuka dan akan hilang segera setelah Anda menutup atau memuat ulang halaman.

2. Penggunaan Data

Data yang Anda masukkan (gambar dan teks watermark) digunakan semata-mata untuk tujuan fungsional aplikasi:

  • Untuk menampilkan pratinjau (preview) di layar Anda.
  • Untuk menggabungkan teks watermark dengan gambar asli.
  • Untuk menghasilkan file gambar baru yang dapat Anda unduh.

Kami tidak menggunakan data Anda untuk analitik perilaku, pelacakan iklan, atau tujuan komersial lainnya.

3. Perlindungan Data

Karena data tidak pernah meninggalkan perangkat Anda, risiko kebocoran data melalui jaringan (Man-in-the-Middle Attack) atau peretasan server dihilangkan sepenuhnya.

  • Isolasi Sandbox: Aplikasi berjalan dalam lingkungan sandbox browser yang aman.
  • Tanpa Database: Kami tidak memiliki database pengguna.

4. Hak Pengguna

Sebagai pengguna, Anda memiliki hak penuh atas data Anda:

  • Hak untuk Tahu: Anda berhak mengetahui bagaimana data Anda diproses (dijelaskan dalam dokumen ini).
  • Hak Kontrol Penuh: Anda memegang kendali penuh atas file asli dan hasil olahan.
  • Tanpa Jejak: Anda dapat menghapus history browser Anda untuk menghilangkan jejak penggunaan aplikasi ini di perangkat lokal Anda.

5. Perubahan Kebijakan

Kami mungkin memperbarui kebijakan privasi ini dari waktu ke waktu untuk mencerminkan perubahan teknologi atau hukum yang berlaku. Perubahan akan diumumkan di halaman ini dengan tanggal pembaruan yang baru.
