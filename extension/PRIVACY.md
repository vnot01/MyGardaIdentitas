# Privacy Policy for GardaIdentitas Chrome Extension

**Last Updated:** January 20, 2026

## 1. Introduction
GardaIdentitas ("we", "our", or "us") respects your privacy. This Privacy Policy describes how the GardaIdentitas Chrome Extension handles user data.

## 2. Data Collection and Usage
**We do not collect, store, share, or transmit any user data.**

*   **Local Processing:** All image processing (watermarking) is performed entirely within your browser (Client-Side) using HTML5 Canvas technology.
*   **No Server Upload:** The photos you select never leave your device. They are not uploaded to any server.
*   **No Analytics:** We do not track your usage or collect behavioral data.

## 3. Permissions Justification
*   **`activeTab`**: Required only to allow the extension to function on the current tab if necessary, although our primary interface is the popup window.
*   **No `host_permissions`**: We do not request access to read or change data on any websites you visit.

## 4. Changes to This Policy
We may update this privacy policy from time to time. Since we do not collect user contact info, we encourage you to check this page for updates.

## 5. Contact
If you have questions about this privacy policy, please open an issue in our GitHub repository.
